# Usage

## Assign

Use **Assign** to apply or remove categories from the currently selected Outlook messages. Multiple selected messages are supported in Outlook classic.

Categories are grouped by macro category. Use the macro category dropdown to narrow the list.

## Manage

Use **Manage** to create, rename, recolor, hide/show, and delete Outlook categories.

Each category can also be assigned to a macro category such as **General** or **Projects**.

## Filter

Use **Filter** to build and run category searches. Select one or more categories, then click **Search**.

Use **All** when messages must match every selected category.

## Settings

Use the gear tab to switch language and manage macro categories.

Deleting a macro category moves its assigned categories back to **General**.
