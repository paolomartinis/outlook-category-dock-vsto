# Changelog

## 1.1 - 2026-05-01

- Added English/Italian UI language support. English is the default.
- Added a Settings tab with a gear icon.
- Added macro category management: create, rename, and delete macro categories.
- Added category grouping by macro category in Assign, Manage, and Filter views.
- Added macro category filters in Assign and Filter.
- Added a click-friendly package installer command file.

## 1.0 - 2026-05-01

- Initial Outlook classic VSTO add-in.
- Added dockable Category Dock task pane.
- Added batch category assignment and removal for selected Outlook messages.
- Added category management: create, rename, color change, hide/show, delete.
- Added category-based search/filter support.
- Added distributable VSTO package.
